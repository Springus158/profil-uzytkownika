# Profil użytkownika

Strona profilu zbudowana w **Next.js 15** + **Tailwind CSS 4**.

Zawiera: avatar, imię i nazwisko, e-mail oraz listę ostatnich aktywności.

## Uruchomienie lokalne

```bash
npm install
npm run dev      # http://localhost:3000
```

## Build / static export

```bash
npm run build    # generuje statyczny katalog out/
```

Wdrożone na GitHub Pages.
